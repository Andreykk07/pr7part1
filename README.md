# Практична робота 7: AI-асистований code review

Цей репозиторій містить результати виконання практичної роботи з проведення систематичного code review за допомогою AI (Варіант 1: E-commerce Order Processing).

## Структура репозиторію

- `original/` — містить файл `orders.js` із початковим legacy-кодом до початку робіт.
- `refactored/` — містить файл `orders.js` із відрефактореним кодом, де виправлено виявлені вразливості, додано транзакції та оптимізовано логіку.
- `tests/` — містить файл `orders.test.js` із 8 characterization тест-кейсами, що фіксують бізнес-вимоги та перевіряють роботу відрефактореного коду.
- `code-review-report.md` — детальний звіт про проведення AI-асистованого review із переліком знайдених проблем та статусом їх виправлення.
